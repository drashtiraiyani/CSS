What is Selectors

CSS Selectors: It defines the pattern to select elements to which css rules can be applied!

Types of selectors

  • Basic Selector
    1. Universal Selector
    2. Class Selector
    3. ID Selector
    4. Element Selector
    5. Attribute Selector
  • Combinator Selector
    1. Descendant
    2. Child
    3. General Sibling
    4. Adjacent sibling
  • Group Selector
  • Pseudo Selector
    1. Pseudo Class Selector
    2. Pseudo Element Selector
