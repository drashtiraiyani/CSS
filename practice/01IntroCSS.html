CSS

What is CSS?

  • CSS=> Cascading Style Sheet

Why we use CSS?

  • CSS=> CSS is used for beautification of our Html Document.
  • CSS=> CSS provides a lot of properties like border, color, background, display, position etc
  • With the help of CSS we can make Animation and Transform

There are 3 ways to apply CSS in our website!

  • Inline CSS

    • We have to apply css under html document itself with style tag.

  • Internal CSS

    • This type says you can add style tag under head section in order to use CSS.

  • External CSS

    • This is the type where you have to make different file with .css extension and link it with html page.
